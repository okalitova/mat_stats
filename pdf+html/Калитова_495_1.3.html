Придумайте распределение, у которого конечны пер- вые четыре момента, а пятое — нет. Это рапределение с плотностью 5 (x ^ (-6)) I([1, INF)).

class mystat(sts.rv_continuous):
    def _pdf(self, x):
        return 5 * (x ** (-6))
stat = mystat(a=1, name='stat')

Сгенерируйте выборку 𝑋1, ..., 𝑋𝑁 из этого распределения для 𝑁 = 10^4.
